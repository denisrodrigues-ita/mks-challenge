## Before all

You need to creat a .env file in root, and copy the .env.exemple content and past there.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

# To ran a test

```bash
npm jest
# or
yarn jest
# or
pnpm jest
# or
bun jest
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
